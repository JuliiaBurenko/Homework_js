2)  null + 3

     ВІДПОВІДЬ: 3. //  В даному випадку null перетворюэться на 0 і виконується дія додавання.


3)  5 && "qwerty"
     
     ВІДПОВІДЬ: qwerty. //виконання оператора "і" проходить до кінця,якщо всі значення true, і повертає останнє обчислене значення. 


4)  +'40' + +'2' + "hillel";
     
     ВІДПОВІДЬ: 42hillel. //Спочатку унарний "+" приводить строки до чисел, далі виконується дія додавання(40+2), а потім конкатинація, бо "hillel" це строка.


5)  '10' - 5 === 6;
    
     ВІДПОВІДЬ: false. //операція віднімання намагається привести строку до числа, тому 10-5 буде 5, а 5 не дорівнює 6, тому отримуємо false.


6)  true + false

     ВІДПОВІДЬ: 1. // при виконанні дії додавання з булевими значеннями  true перетворюється на 1, а false на 0, і виконується дія додавання (1+0).
 

7)  '4px' - 3

     ВІДПОВІДЬ: Nan. // так як першим операндом є рядок, який не приводиться до чесла, то дія віднімання не може виконатись, відповідь - Not A Number.


8)  '4' - 3

     ВІДПОВІДЬ: 1. // операція віднімання приводить строку до числа, якщо це можливо, а потім виконується віднімання(4-3).


9)  '6' + 3 ** 0;

     ВІДПОВІДЬ: 61. // операція зведення до степеня є правоасоціатиною, тому спочатку виконується дія 3**0, це дорівнює 1, а потім виконується конкотинація, бо '6'-це строка.


10)  12 / '6' 

     ВІДПОВІДЬ: 2. // операція ділення намагається привести строку до числа(якщо це можливо), а потім виконати дію (12/6).


11)  '10' + (5 === 6);

     ВІДПОВІДЬ: 10false. // спочатку виконується дія в дужках, отримуємо false, далі, т.я. першим операндом є строка, то відбувається конкотинація двох строк "10" і "false".

13)  3 ** (9 / 3);

14)  !!'false' == !!'true'

     ВІДПОВІДЬ: true. // перше заперечення перетворює строку на булеве значення false, а друге - на true, не строга рівність дає true.


15)  0 || '0' && 1

     ВІДПОВІДЬ: 1. // оператор "або" повертає true, коли хочаб одине зі значень є true('0'-це строка), а операція 'і' проходить до кінця, коли всі true і виводить останнє обчислене значення.


16)  (+null == false) < 1;

     ВІДПОВІДЬ: false. // унарний '+'  приводить null до числа 0,  false також має значення 0, 0 == 0 дає true, true має значення 1, тоді 1 < 1 повертає false. 


17)  false && true || true

     ВІДПОВІДЬ: true. // опервції "і" та "або" є лівоасоціативні, тому спочатку виконуємо false && true, отримуємо false, потім false || true, отримуємо - true. 


18)  false && (false || true);

     ВІДПОВІДЬ: false. // спочатку виконується операція в дужках, отримуємо true, потім false && true, отримуємо false.


19)  (+null == false) < 1 ** 5;

     ВІДПОВІДЬ: false. // унарний '+'  приводить null до числа 0,  false також має значення 0, 0 == 0 дає true, true має значення 1, з іншого боку 1 ** 5 є 1, тоді 1 < 1 повертає false.
